# Политика конфиденциальности — ProxyPilot

_Последнее обновление: 2026-06-10_

ProxyPilot — расширение для браузера Chromium, которое маршрутизирует выбранные
пользователем домены через прокси-сервер. Эта политика описывает, какие данные
расширение обрабатывает.

## Коротко

**ProxyPilot не собирает, не хранит на стороне сервера и не передаёт третьим
лицам никаких персональных данных, истории просмотров или содержимого страниц.**

## Какие данные обрабатываются и где

Все данные хранятся **локально** в браузере пользователя (`chrome.storage`) и не
покидают устройство:

- Настройки: выбранные сервисы/домены для маршрутизации, выбранный источник
  прокси, адрес и учётные данные вашего прокси (если вы их ввели), тема оформления.
- Технический кэш: список публичных бесплатных прокси и реестр заблокированных
  доменов Роскомнадзора — для работы без постоянных загрузок.

Учётные данные вашего прокси используются исключительно для подключения к этому
прокси и хранятся только локально.

## Сетевые запросы

Расширение выполняет внешние запросы только для своей работы:

- Загрузка публичных списков бесплатных прокси (GitHub/CDN) — это данные
  конфигурации, не данные пользователя.
- Загрузка реестра заблокированных доменов (для соответствия 149-ФЗ).
- Небольшие тестовые запросы для проверки работоспособности прокси.

Эти запросы не содержат вашу личную информацию.

## Содержимое страниц

Расширение **не читает, не изменяет и не передаёт** содержимое посещаемых вами
страниц. Разрешение на доступ ко всем сайтам используется только для применения
правил маршрутизации (PAC), поскольку список маршрутизируемых доменов задаёте вы.

## Бесплатный пул прокси

Бесплатные публичные прокси управляются третьими лицами и не контролируются
разработчиком. Не используйте их для входа в важные аккаунты. Разработчик не несёт
ответственности за работу сторонних прокси-серверов.

## Аналитика и трекинг

Расширение **не использует** аналитику, рекламные сети, файлы cookie для трекинга
или иные средства слежения.

## Изменения

При изменении политики дата обновления вверху будет изменена.
